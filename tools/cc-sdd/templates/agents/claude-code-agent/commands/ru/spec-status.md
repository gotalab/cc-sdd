# Статус спецификации

<background_information>
- **Миссия**: Отобразить полный статус и прогресс спецификации по всем фазам
- **Критерии успеха**:
  - Чёткий обзор всех фаз
  - Процент завершения для каждой фазы
  - Рекомендации по следующим шагам
</background_information>

<instructions>
## Основная задача
Показать полный статус спецификации для функции из $ARGUMENTS.

## Шаги выполнения
1. **Загрузить метаданные**: `{{KIRO_DIR}}/specs/{feature}/spec.json`
2. **Проанализировать файлы**:
   - requirements.md: количество требований
   - design.md: количество компонентов
   - tasks.md: выполненные/всего задач
3. **Вычислить проценты** завершения
4. **Определить блокеры** и следующие действия
</instructions>

## Описание вывода
```
# Статус: {feature-name}

## Обзор функции
**Имя**: {name}
**Фаза**: {current-phase}
**Язык**: ru

## Статус фаз

### ✅ Требования (100%)
- Сгенерировано: Да
- Согласовано: Да
- Всего: X требований, Y критериев

### ✅ Дизайн (100%)
- Компоненты: X
- Диаграммы: Y

### ⏳ Реализация (58%)
- Завершено: 14/24 задач
- Осталось: 10 задач

## Следующее действие
/kiro:spec-impl {feature} 4.3,4.4,5.1
```
