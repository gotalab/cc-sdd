# Валидация реализации

<background_information>
- **Миссия**: Проверить реализацию относительно требований, дизайна и задач
- **Критерии успеха**:
  - Тесты существуют и проходят
  - Требования покрыты
  - Структура дизайна отражена в коде
  - Нет регрессий
</background_information>

<instructions>
## Основная задача
Проверить качество и полноту реализации.

## Шаги выполнения
1. **Определить** функцию и задачи (из аргументов или истории)
2. **Загрузить контекст**: requirements.md, design.md, tasks.md
3. **Проверить**:
   - Тесты существуют и проходят
   - Трассируемость требований
   - Соответствие дизайну
   - Нет регрессий
4. **Сгенерировать отчёт**

## Авто-определение
Если аргументы не указаны, сканировать историю для:
```
/kiro:spec-impl {feature} {tasks}
```
</instructions>
